JavaScript全栈随笔（一）

一、简介

JavaScript是世界上流行的脚本语言,网页的交互逻辑都是由JavaScript驱动的.即JavaScript是一种运行在浏览器的解释型编程语言。优点：在Web中JavaScript跨平台、跨浏览器驱动网页，与用户进行交互。

二、背景

为了让JavaScript成为全球统一的标准，几个公司联合ECMA组织制定了JavaScript语言标准，被称为ECMAScript标准。2015年6月发布最新版本的ECMAScript 6简称（ES6），许多新特性IE6以下版本是不支持的。

三、使用

1、直接嵌在网页，采用包裹，一般放在之中.

2、将Javascript单独放在.js文件中，在html中引用该文件.便于维护及多个页面各自引用一份js文件

四、调试

在浏览器使用开发者模式（F12）进行调试。1、函数中加入断点debugger,F10执行下一步、F11进入方法
2、通过控制台打印console.log(‘123’)或弹出框alert(‘456’);
